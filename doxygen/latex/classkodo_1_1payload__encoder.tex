Encodes a symbol to the provided buffer using the following layout\-:


\begin{DoxyCode}
+-------------------+---------------+
|    symbol data    |   symbol \textcolor{keywordtype}{id}   |
+-------------------+---------------+
\end{DoxyCode}


The reason the symbol data is placed first in the payload buffer is to increase the chances of 16 byte-\/alignment of the symbol data. If the variable length id would be place in front of the symbol it would easily become unaligned. Unaligned symbol data access will most likely result in very bad performance.Encodes a symbol into the provided buffer.
