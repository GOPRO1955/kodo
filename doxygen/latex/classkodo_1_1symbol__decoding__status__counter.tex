\subsection{Detailed Description}
\subsubsection*{template$<$class Super\-Coder$>$class kodo\-::symbol\-\_\-decoding\-\_\-status\-\_\-counter$<$ Super\-Coder $>$}

The symbol decoding status counter maintains information about how many symbols are in which state. 

On the decoder side there are three states a symbol can be in information about these states can be found in the \hyperlink{classkodo_1_1symbol__decoding__status__tracker}{symbol\-\_\-decoding\-\_\-status\-\_\-tracker} layer. This layer maintains counters showing how many symbols are in each state.
